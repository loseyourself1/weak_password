# 密码强度检测浏览器插件

本插件用于在用户登录或注册时，实时检测密码是否符合复杂度要求。

## 功能特点

- 实时检测密码强度，当用户输入密码时即时反馈
- 检测密码是否满足以下条件：
  - 长度大于8个字符
  - 包含小写字母
  - 包含大写字母
  - 包含数字
  - 包含特殊符号（如 !@#$%^&*() 等）
- 支持动态加载的密码输入框
- 提供直观的视觉反馈，包括提示信息和输入框边框颜色变化

## 安装方法

### Chrome / Edge 浏览器

1. 打开浏览器，在地址栏输入 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
2. 打开右上角的「开发者模式」开关
3. 点击「加载已解压的扩展程序」按钮
4. 选择本项目所在的文件夹
5. 插件已安装成功，可以在浏览器工具栏中看到插件图标

## 使用方法

安装成功后，插件会自动在所有网页上工作：

1. 当你在任何网站的密码输入框中输入密码时，插件会实时检测密码强度
2. 如果密码不符合要求，输入框边框会变为红色，并显示具体缺少的要求
3. 如果密码符合要求，输入框边框会变为绿色，并显示「密码强度符合要求」的提示

## 技术说明

- 插件使用Manifest V3标准开发
- 使用JavaScript实现密码强度检测逻辑
- 使用MutationObserver监听页面变化，支持动态加载的密码输入框
- 所有图标使用SVG格式，确保在高分辨率屏幕上清晰显示

## 注意事项

- 插件只会在本地检测密码强度，不会上传或存储你的密码信息
- 目前插件仅提供提示功能，不会阻止表单提交
- 如果遇到兼容性问题，请尝试刷新页面或重新安装插件
